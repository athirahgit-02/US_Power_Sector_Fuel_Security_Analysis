# Strategic Analysis of US Power Sector Fuel Resilience  

### By Nurul Athirah  

## Business Problem  
As the energy transition accelerates, how secure are regional electricity grids? This project analyzes fossil-fuel stock volatility to identify supply risks in the US power sector.  

## Data Sourcing  

Source: U.S. Energy Information Administration (EIA) Electricity Data (Form EIA-923).  

Data Composition:  
1. Electric Power Sector
2. State
3. Census Division
4. Coal and Rank
